Cadastro de Vendas

Suporte de Cadastro de Vendas
Para realizar o cadastro de uma venda, você deverá seguir os seguintes passos:
• Selecionar a opção do tipo de venda (geral ou encomenda).
• Preencher os campos de acordo com a opção selecionada (exemplo: se o tipo de venda selecionada for geral,deverá preencher os campos de vendedor, data, total geral).
• Clicar na seta para prosseguir para a parte de forma de pagamento e preencher os campos.
• Para finalizar, você deverá clicar no botão cadastrar.
• Caso deseje sair desta página, clique no "x" acima.
